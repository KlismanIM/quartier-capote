## 📌 Visão Geral

Landing page institucional desenvolvida para um empreendimento de alto padrão localizado entre **Jardins e Pinheiros**, em São Paulo.

**Foco estratégico:**
- Posicionamento premium
- Experiência visual refinada
- Conversão de leads qualificados

---

## 🎯 Objetivo de Negócio

- Apresentar o empreendimento com linguagem de luxo e exclusividade  
- Valorizar arquitetura assinada por **Königsberger Vannucchi**  
- Destacar diferenciais competitivos (paisagismo, interiores, localização)  
- Direcionar leads diretamente para especialista  
- Maximizar conversão via formulário e WhatsApp  

---

## 🧠 Funcionalidades

- Design responsivo (mobile → 4K)  
- Menu dinâmico com estado ativo  
- Scroll tracking inteligente  
- Hero com animação de entrada  
- Carrossel interativo (Swiper)  
- Vídeo institucional com proteção  
- Formulário integrado (Formspree)  
- WhatsApp flutuante com CTA direto  
- Lazy loading e otimização de performance
