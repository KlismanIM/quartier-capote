📌 Visão Geral
Desenvolvimento de uma landing page institucional responsiva de alto padrão para o empreendimento Quartier Capote, localizado entre os nobres bairros dos Jardins e Pinheiros em São Paulo.

🤖 Abordagem de Desenvolvimento
Este projeto foi desenvolvido com uma abordagem híbrida e moderna:

Etapa	Ferramenta	Contribuição
Arquitetura base	IA (Claude/GPT)	Geração da estrutura inicial, componentes modulares e organização de pastas
Refinamento técnico	Manual	Ajustes finos de layout, correções de responsividade, otimização de performance
Integração	Manual	Configuração de bibliotecas (Swiper, Alpine, Lucide), testes cross-browser
Versionamento	Manual	Commits semânticos, organização do histórico, documentação
Por que essa abordagem?

✅ Velocidade de desenvolvimento sem sacrificar qualidade

✅ Código final revisado e refinado por um engenheiro frontend

✅ Aprendizado contínuo e documentação do processo

✅ Resultado final com padrão profissional e escalável

A solução foi construída com arquitetura modular moderna, priorizando:
✔ Performance e carregamento otimizado
✔ Experiência de luxo (UX/UI refinada)

## 🧠 Funcionalidades

- ✨ **Design responsivo** — Mobile-first com breakpoints de 375px a 1536px *(IA + Refino)*  
- ✨ **Menu dinâmico** — Controle de estado e seção ativa com Alpine.js *(IA + Refino)*  
- ✨ **Scroll tracking** — Detecção automática da seção visível *(IA)*  
- ✨ **Hero animada** — Efeito float e slide-up na entrada *(IA)*  
- ✨ **Galeria interativa** — Carrossel Swiper com autoplay *(IA)*  
- ✨ **Vídeo institucional** — Player customizado *(Refino manual)*  
- ✨ **Formulário funcional** — Integração com Formspree *(IA)*  
- ✨ **WhatsApp flutuante** — CTA direto *(IA)*  
- ✨ **Recomendações inteligentes** — Cooldown de 50s *(Refino manual)*  
- ✨ **Smooth scroll** — Navegação fluida *(IA)*  
- ✨ **Lazy loading** — Otimização de imagens *(Refino manual)*  
- ✨ **Touch optimizations** — Mobile UX refinado *(Refino manual)*
